# NUS Mars Rover Team Website

Official website for the NUS Mars Rover Team — a student engineering team at the National University of Singapore competing in the University Rover Challenge (URC).

## Tech Stack

- [Next.js](https://nextjs.org/) 15 (App Router)
- TypeScript
- Tailwind CSS v4

## Getting Started

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
Mars-Rover-Website/
├── frontend/        # Next.js app
│   ├── app/         # Pages and layouts
│   └── public/      # Static assets
└── backend/         # (coming soon)
```

## Editing Website Content

All website content is stored as JSON files inside `frontend/data/`. You do not need to touch any code — just edit the JSON and add or remove images in the relevant folder.

Open the README inside each folder for step-by-step instructions:

| Folder | Controls |
|--------|----------|
| [`frontend/data/contact/`](frontend/data/contact/README.md) | Email address, Instagram, LinkedIn, YouTube links |
| [`frontend/data/aboutus/`](frontend/data/aboutus/README.md) | Mission, vision, outreach goals and outreach photos |
| [`frontend/data/homepage/`](frontend/data/homepage/README.md) | Hero text, stats, rover gallery images |
| [`frontend/data/events/`](frontend/data/events/README.md) | News entries and event photos |
| [`frontend/data/publications/`](frontend/data/publications/README.md) | GitHub link, PDR/SAR reports |
| [`frontend/data/rovers/`](frontend/data/rovers/README.md) | Rover writeups, SAR video, subteam sections and images |
| [`frontend/data/team/`](frontend/data/team/README.md) | Team members by year (leadership + subsystems) and photos |
